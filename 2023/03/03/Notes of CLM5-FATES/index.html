FATES and CLM5-FATES

FATES是一个陆地生态系统模型，但是需要作为额外的模块搭载在不同的宿主陆面模式上，目前支持搭载在CESM、CLM和E3SM上。

FATES( Functionally Assembled Terrestrial Ecosystem Simulator) is a numerical terrestrial ecosystem model and an external module which can run within a given “Host Land Model” (HLM).

FATES is a module, designed to operate insides a ‘host’ land surface model, that simulates vegetation physiology, growth, competition, ecosystem assembly and vegetation distribution. FATES is a ‘vegetation demographics model’ that uses a cohortized representation of plant populations to approximate ecosystem behaviour.

FATES脱胎于应用ED071[0557:AMFSVD]2.0.CO;2)概念于CLM4.5上的CLM(ED)模型。CLM5不再支持之前版本的DGVM模块，而是引入了FATES作为新的动态全球植被模块，作为CLM5-FATES。

CLM5和FATES都仍在发展，尚未总结最新的FATES发展到了何等程度。但是从CLM5和FATES的技术文档来看，CLM5-FATES是进行了简化的。比如碳分配，FATES有不同的假说和实现，CLM5-FATES只支持第一种假说。

The implementation of the Ecosystem Demography concept within FATES links the surface flux and canopy physiology concepts in the CLM/E3SM with numerous additional developments necessary to accommodate the new model also documented here.

Case 1 - A Simple Case

在巴西上运行5年的I2000Clm50FatesCruGs的单点模拟，以月输出，不需要

从任意位置冷启动，起转五年。

  • This script will BUILD a CESM single point simulation for the 1x1_brazil resolution.
  • 1x1_brazil is a pre-made “resolution”, meaning the support files domain/surface/etc are already available.
  • This is a single-site, located roughly in the southern Amazon.
  • CLM_FORCE_COLDSTART when set to on, _requires_ that your simulation do a cold start from arbitrary initial conditions. If this is NOT set, it will use an initial condition file if it can find an appropriate one, and otherwise do a cold start. CLM_FORCE_COLDSTART is a good way to ensure that you are doing a cold start if that is what you want to do. The “on” value forces the model to spin up from a cold-start (arbitrary initial conditions).
  • DATM_CLMNCEP_YR_START sets the beginning year to cycle the atmospheric data over for CLM_QIAN or CLMCRUNCEP modes.
  • DATM_CLMNCEP_YR_END sets the ending year to cycle the atmospheric data over for CLM_QIAN or CLMCRUNCEP modes.

Does FATES work globally?

Yes, and, No. FATES can be and is run globally. We are working on a scientifically supportable plant functional type characterisation for ‘standard’ global runs at present.
