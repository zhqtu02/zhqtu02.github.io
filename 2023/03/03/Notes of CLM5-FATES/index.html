FATES and CLM5-FATES

FATES是一个陆地生态系统模型，但是需要作为额外的模块搭载在不同的宿主陆面模式上，目前支持搭载在CESM、CLM和E3SM上。

FATES( Functionally Assembled Terrestrial Ecosystem Simulator) is a numerical terrestrial ecosystem model and ==an external module which can run within a given “Host Land Model” (HLM)==.

FATES脱胎于应用ED071[0557:AMFSVD]2.0.CO;2)概念于CLM4.5上的CLM(ED)模型。CLM5不再支持之前版本的DGVM模块，而是引入了FATES作为新的动态全球植被模块，作为CLM5-FATES。

CLM5和FATES都仍在发展，尚未总结最新的FATES发展到了何等程度。但是从CLM5和FATES的技术文档来看，CLM5-FATES是进行了简化的。比如碳分配，FATES有不同的假说和实现，CLM5-FATES只支持第一种假说。

The implementation of the Ecosystem Demography concept within FATES links the surface flux and canopy physiology concepts in the CLM/E3SM with numerous additional developments necessary to accommodate the new model also documented here.
